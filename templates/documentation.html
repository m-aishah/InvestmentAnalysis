API Documentation

Introduction

This API provides access to various investment analysis features including cost comparison, property details and insights, rental income forecasts, and risk analysis.

It is built mainly as an extension for CMND.ai but includes some standalone endpoints.

Endpoints

Cost Comparison POST

Compare property costs based on detailed criteria including budget, location, size, and more

Property Details POST

Retrieve detailed information about a specific property

Rental Income Forecast POST

Get a forecast of potential rental income for properties

Risk Analysis POST

Evaluate the risk associated with a property

Investment RecommendationPOST

Recommend porperties based on overall information

CMND TOOLS GET

See details about tools available for CMND.ai

Run CMND TOOLS POST

Used by CMND to run tools.

CMND.ai Integration

To integrate this API with CMND.ai, follow these steps:

  1. Make a connection on CMND.ai with the URL -> http://13.61.32.149/
  2. ...and start a conversation!

For more info, please refer to our CMND.ai Guide.

Cost Comparison

Endpoint: /api/cost-comparison

Method: POST
Request Body:

                        {
                            "budget_min": 0,
                            "budget_max": 1000000,
                            "location": "string",
                            "size_min": 0,
                            "size_max": 0,
                            "bedrooms_min": 0,
                            "bedrooms_max": 0,
                            "bathrooms_min": 0,
                            "bathrooms_max": null,
                            "family_size": null,
                            "property_type": "string",
                            "sort_by": "string"
                        }
Parameters:
Field Type Description Required
budget_min integer Minimum budget for property investment No
budget_max integer Maximum budget for property investment Yes
location string Location preference for property investment No
size_min integer Minimum property size in square meters No
size_max integer Maximum property size in square meters No
bedrooms_min integer Minimum number of bedrooms No
bedrooms_max integer Maximum number of bedrooms No
bathrooms_min float Minimum number of bathrooms No
bathrooms_max float Maximum number of bathrooms No
family_size integer Number of family members No
property_type string Type of property (e.g., house, apartment) No
sort_by string Sort the results by a specific field (e.g., price, size) No
Response:

Returns a list of properties with their cost comparisons based on the provided criteria.
